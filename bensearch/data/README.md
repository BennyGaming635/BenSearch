## READ THIS BEFORE ADDING TO THE COMMUNITY JSON
We aim to have a clutter-free dataset to avoid duplicates and extra content in the database that is not needed. When adding data, please confirm that it;

### 1. Does not already exist
Make sure that what you want to add does not already exist. If it exists maybe modify the description or add to the filters...

### 2. Isn't 'clutter'
Make sure your site that you are adding isn't something like a verification page, login page, language-modifed wikipedia page etc, as these cause clutter not just for you but everyone else who uses our services, and it's not fair for them.

### 3. It's safe.
Finally, make sure that the link you want to add is safe and doesn't lead to a 'sketchy' place which could advertise adult content, drugs, gambling, propaganda, viruses, crimes (including piracy) etc.

At any moment, we may place a link on a blacklist, which will not allow adding it, in addition to this, at any moment we can decide that a link **will** be removed from our database for reasons including but not limited to the above.

Thank you for contributing to BenSearch!
